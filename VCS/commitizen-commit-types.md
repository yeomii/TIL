# Commitizen cli 에서 제안하는 commit message types

* https://github.com/commitizen/cz-cli

## Types
* feat
    * 새로운 기능
* fix
    * 버그 수정
* docs
    * 문서만 변경
* style
    * 코드 자체의 의미에 영향이 없는 변경
    * 공백, 포맷팅, 빠진 세미콜론 추가 등
* refactor
    * 버그 수정도 아니고 기능 추가도 아닌 코드 변화
* perf
    * 성능을 향상시키는 코드 변화
* test
    * 빠진 테스트 코드 추가
* chore
    * 빌드 프로세스나 보조하는 툴, 또는 문서 생성과 같은 라이브러리 등을 변경
